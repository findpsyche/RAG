## ❓ 遇到问题的解决方案

### 问题: "我对 Embedding 的原理还不够理解"
**解决方案**:
1. 重新读 RAG_Day1_Core_Concepts.md 的 Embedding 部分
2. 在 25_12_22.py 里修改 simple_hash_embed 函数，看看不同的参数怎么影响相似度
3. 运行时加上 print 语句，看向量的具体值

### 问题: "不知道怎么改 Demo 用自己的文档"
**解决方案**:
1. 把你的文档放在 sample_docs/ 目录
2. 运行 RAG_Day3_Demo.py 时，看它怎么加载文档的
3. 修改 DocumentLoader.load_markdown_files() 的路径即可
4. 如果是 PDF，用 PyPDF2 库先提取文本

### 问题: "我给客户做 Demo，他问了没有在清单里的问题"
**解决方案**:
1. 这很正常，每个公司都不一样
2. 记录这个问题
3. 查看 RAG_Day2_Business_Adaptation.md 的相关章节
4. 基于 5 问诊断法来回答这个问题

### 问题: "Demo 运行出错"
**解决方案**:
1. 确保 Python 版本 >= 3.7
2. 检查代码中的中文注释是否正确编码
3. 查看 sample_docs/ 文件是否生成成功
4. 如果还是有问题，简化 Demo 代码，逐步排查
