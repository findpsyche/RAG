# 我们的产品

## 产品功能
- AI 中台: 提供企业级的大模型接口和向量数据库
- 知识库系统: 支持 PDF、Word、Excel 等多种格式
- 智能问答: 基于 RAG 技术的企业知识问答系统

## 技术栈
- 后端: Python + FastAPI
- 数据库: PostgreSQL + Milvus (向量数据库)
- 部署: Docker + Kubernetes
- 前端: React + TypeScript
